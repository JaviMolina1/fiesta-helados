# 🍦 Fiesta Helados

Aplicación web desarrollada con **Django** para la gestión y venta de productos de una heladería.

El proyecto fue desarrollado originalmente como parte de un proyecto académico de Ingeniería en Informática. La aplicación permite diferenciar entre clientes y administradores, gestionar productos y realizar un proceso de compra mediante un carrito.

## Funcionalidades

- Registro e inicio de sesión de usuarios.
- Diferenciación entre usuarios clientes y administradores.
- Visualización de productos disponibles.
- Carrito de compras.
- Modificación de cantidades en el carrito.
- Generación del detalle de compra.
- Administración de productos.
- Creación de nuevos productos.
- Actualización de productos existentes.
- Eliminación de productos.
- Carga de imágenes para productos.
- Edición del perfil de usuario.

## Tecnologías utilizadas

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Estado del proyecto

El proyecto se encuentra funcional y actualmente está siendo revisado y mejorado para fortalecer su estructura, documentación y experiencia de usuario.
